README – CSS Hover Effects & Transitions
 Project Title

CSS Hover Effects and Transitions Demo

 Description

This project demonstrates basic CSS hover effects and transitions using a simple HTML webpage.
It shows how elements like buttons, images, and cards respond when the user moves the mouse over them.

 Features

 Button hover effect with color change and scaling

 Image hover effect with rotation and zoom

 Card hover effect with upward movement and shadow

 Smooth animations using CSS transition property

 Technologies Used

HTML5 – Structure of the webpage

CSS3 – Styling, hover effects, and animations

 How It Works
 Button Effect

Changes color from blue to green

Slightly increases in size using scale()

 Image Effect

Rotates slightly using rotate()

Zooms in using scale()

 Card Effect

Moves upward using translateY()

Shadow becomes stronger for a 3D effect
